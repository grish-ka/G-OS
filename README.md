# G-OS

is a wip(work in progress) os.

## Setup & Running
1. move all the files to linux([ubunto](https://ubuntu.com/download/desktop) is what I used)
2. ```bash
    $ sudo apt install qemu-system-x86
    ```
3. ```bash
    $ ./run.sh
    ```

| Software | Has support | Rateing |
| :-------:| :---------: | :-----: |
| QEMU     | True        | 5       |
| VMWare   | False       | 0       |
